## Projects

### 🔬 Breast Cancer Image Classification — Custom CNN vs Transfer Learning (MSc Thesis)
**Dataset:** 277,524 histopathology image patches from 162 patients (Kaggle Breast Histopathology Images)  
**Objective:** Designed and evaluated a custom CNN architecture against four pre-trained transfer learning models for invasive ductal carcinoma (IDC) detection.

**Key Results:**

| Model | F2-Score | Recall | Accuracy | ROC-AUC | Training Time |
|---|---|---|---|---|---|
| **Custom CNN (mine)** | **0.8684** | **87.79%** | **83.27%** | **0.9239** | **4.2 min** |
| DenseNet201 | 0.8228 | 82.15% | 82.56% | 0.9016 | 15.2 min |
| ResNet50 | 0.8302 | 83.43% | 82.20% | 0.9052 | 4.8 min |
| VGG16 | 0.7876 | 77.89% | 80.65% | 0.8793 | — |
| MobileNetV2 | 0.7732 | 76.21% | 79.79% | 0.8706 | 3.7 min |

**Contributions:**
- Designed a custom CNN architecture from scratch, tailored to 50×50 pixel H&E-stained tissue patches
- Achieved 5–8% improvement over all transfer learning baselines across F2-Score, ROC-AUC, and MCC
- Detected 6,584 of 7,500 cancer cases — 327 more than second-best ResNet50
- Reduced false negatives by 26.3% vs ResNet50 and 48.6% vs MobileNetV2
- Trained 3.6x faster than DenseNet201 (4.2 min vs 15.2 min)
- Applied Grad-CAM interpretability analysis across all architectures
- Implemented hybrid CNN-SVM approach with RBF kernels
- Validated results using McNemar test and bootstrap resampling (95% CI, 1,000 iterations)
